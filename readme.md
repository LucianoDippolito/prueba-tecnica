# Prueba Técnica: Lector de Recetas Médicas Digitales

## Objetivo

Desarrollar una solución que sea capaz de leer y extraer el contenido de recetas médicas digitales en formato PDF.

## Descripción del Proyecto

Se requiere implementar una aplicación que:

- **Lea archivo PDF** que contenga una receta médica digital
- **Extraiga información relevante** como:
  - Datos del paciente
  - Medicamentos prescritos
  - Indicaciones y contraindicaciones
  - Fecha de emisión y validez
  - Otros
- **Procese la información** de manera estructurada
- **Presente los resultados** de forma clara y organizada

## Requisitos Técnicos

- Lenguaje de programación: A tu elección, preferentemente python
- Capacidad de manejo de archivos PDF
- Extracción y parseo de datos

## Entregables

- Código completamente funcional
- Documentación de instalación y uso
- Ejemplos de ejecución
- Explicación como se llego a la o las solución/es
